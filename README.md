# Python+Vue3 文件分享管理系统 python387

## 项目概述

本项目是一个基于 Python 和 Vue3 的前后端分离文件分享管理系统，包含了详尽的源码、数据库文件和部署文档。代码均为原创手写，核心部分带有注释，页面设计美观精致。

## 技术栈

- 前端：Vue3
- 后端：Python（3.8版本）
- 数据库：MySql（8.x版本）
- Node版本：v20.11

## 功能模块

### 用户功能

- 用户注册与登录
- 密码找回邮件通知
- 账号页面权限分离
- 用户首页：展示登录信息与系统公告通知
- 个人信息修改
- 文档管理：
  - 文档上传与批量上传
  - 设置文档分类与访问权限（公开或私密）
  - 查询、预览及下载公开文档
  - 编辑和删除自己上传的文档
  - 分享私密文件给他人（含有效期和提取密码设置）

### 管理员功能

- 数据大屏：数据可视化展示
- 文档分类管理：增删改查
- 数据中心：
  - 管理所有用户上传的文档
  - 筛选、查看、编辑、删除文档信息
  - 导出数据为Excel
- 角色管理：自定义角色与权限
- 用户管理：增删改查、拉黑操作
- 公告管理：自定义系统公告
- 系统日志：运行日志与安全审计日志
- 系统资源监控：CPU、内存、网络流量实时监控

## 系统角色

- 用户：可管理个人信息和上传的文档
- 管理员：全面管理系统中所有文档和用户信息
